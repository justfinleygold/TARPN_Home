TARPN Home

This software is designed as a simple and useful interface to a TARPN node. It's 4 tabs each have a separate connection to the node, so you can use them independently. It should run in ANY browser running on the same Wi-fi as your node. Just enter the local ip address (ex: 192.168.x.xxx) of your node followed by :8085

The Chat tab lets you join the group chat. The Node tab lets you enter node user commands. The BBS tab lets you see, read, and send BBS messages. The Info tab lets you see node information.

  • Features

  • -> Connect from any computer, phone, or tablet through Wi-fi
  • -> Shows traffic on all connected devices automatically
  • -> Beeps if chat messages arrive after 5 minutes of inactivity
  • -> Beeps if chat messages arrive and window is not focused
  • -> Maintains connection to the node even when web clients are gone
  • -> Provides a simple interface to the node BBS
